Project Summary:

The analysis that will be described below was done for a 2013 final project during my undergraduate course for Advanced GIS. This project started out with the idea that Salt water crocodiles spend more time in one type of cover type than another. The goal for this project was to discover what cover type they spend more time.

Methods: Raster Analysis

1. Stack bands 3 and 7 from Landsat data

2. Use Radiance Model in Erdas Imagine to correct for radiace. Stack 3 rasters

3. Run Tassled Cap Analysis (separates brightness and greenness values to make it simpler to separate Wetlands from Savannah).

4. Mosaic images together, for further analysis.

Methods: Homerange Analysis

1. Create Kernel Rasters for each crocodile.

2. Create Isopleths for the 95%, 90%, 85% and 80% home range size. To see if there was a difference in home range sizes and also in the use and availability values.

Methods: Use Vs. Availability Analysis

1. Reclassify moasicked raster into 4 classifications, Wetlands, Savanah, Forest, Water.

2. Convert Isopleths to polygons.

3. Clip raster to polygons

4. Use Tabulate Area (ArcMap)to calculate availability of the land cover

5. These values were all divided by 900 to acquire the pixel value per home range. These values were then imported to excel and formed into Bard Graphs.
